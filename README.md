## INTRODUCTION

#### WHAT IS THE PROJECT
- This is a simple project coded completely in C++

- It's a game which deals with running a governmnet and alsp trying to dominate the world parallelly.
- The player may need to eliminate all other nations/countries in the game and at the same time, may also need to work on keeping the citizens satisfied.
- The game also involves real time gameplay and duration for creating various things similar to an actual government.
